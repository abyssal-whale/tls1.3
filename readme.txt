TLS1.3项目使用说明：
1、解压文件后会看到一个名为PJ1的文件夹
2、进入PJ1文件夹，会看到client.py和server.py两个文件
3、使用命令行分别运行两个.py文件（一定先运行server后运行client）
     指令分别为“python server.py”和“python client.py”
4、命令行打开server后看不到反应（这才是正常反应）
5、打开client后根据提示输入用户输入的内容，
     并从DES和DES3中选择一种作为加密套件（只需要输入DES或者DES3）
6、稍等片刻后即可在运行server.py的命令行最底端看到输出，
     即为解密后得到的用户输入
7、重中之重，如果遇到报错或者卡住，可以重新跑一遍（那一定是你运气太好了，手动哭脸）
8、如果再跑一遍还是不行的话可能需要麻烦助教换一个python版本试一下（手动玫瑰）
